# phpStorm

### 启动时间

```bash
# 查看 IDE启动插件占用时间，可以关掉不必要的插件
Help -> diagnost Tools -> Analyze Plugin Startup Performance
```

### 设置内存
```bash
# 电脑16G 一般我都设置成4096（4G）
Help -> Change Memory Setting
```
### 快捷键

```bash
command + option + l       #格式化代码
command + delete           #删除当前行
command + d                #复制当前行
command + w                #关闭当前tab
command + shif + []        #切换文件tab
command + b                #跳转到变量声明处
command + shif + c         #复制当前文件绝对路径
command + shif + u         #大小写转换
command + shif + +/-       #代码折叠展开
command + shif + v         #打开粘贴板
shift 按两次                #全局搜索
shift + option + 鼠标       #选中多行
command + 1                #显示隐藏左侧文件
command + 9                #显示代码提交记录
command + 0                #显示待提交的代码
command + ↑                #导航栏操作
command + k                #Git提交
command + d                #git文件diff
```
