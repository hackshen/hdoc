## [vConsole](https://github.com/Tencent/vConsole)

> 一个轻量、可拓展、针对手机网页的前端开发者调试面板。

#### 特性

- 查看 console 日志
- 查看网络请求
- 查看页面 element 结构
- 查看 Cookies、localStorage 和 SessionStorage
- 手动执行 JS 命令行
- 自定义插件
